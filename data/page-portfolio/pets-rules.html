Pets Rules

A casual mobile game in which the player assumes the role of a dog or cat in a procedurally generated house, with the goal of completing quests focused on making the biggest mess possible while avoiding to displease the human of the house.

In this game I worked writing code on:
• UI
• Gameplay and Animation
• Backend and saving system using Playfab

I also worked on specific game systems such as:
• Shop and customization
• Game map generation
• Tutorial system
• Quest system
• Procedural generation system (seed management)
The game was made with Unity, scheduled for launch in the second half of 2023.
