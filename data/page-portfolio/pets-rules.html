Pets Rules

A casual mobile game in which the player assumes the role of a dog or cat in a procedurally generated house, with the goal of completing quests focused on making the biggest mess possible while avoiding to displease the human of the house. The game was developed in Unity for PC and mobile at Umbu Games.

In this game I worked writing code on:
  • • Gameplay Systems
  • • Gameplay and Menu UI
  • • Shop and customization
  • • Game map generation
  • • Tutorial system
  • • Quest system
  • • Procedural level generation system (seed creation/load/storage)
  • • Backend integration and saving system using Playfab
