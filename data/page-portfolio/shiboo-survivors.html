Shiboo Survivors

Inspired by Vampire Survivors. Shiboo Survivors is a Roguelike WebGL game where you play as a Shiba Inu, battling wave enemies across different levels until time runs out. The game allows playng solo or coop mode with friends and players worldwide in multiplayer mode. Developed in Unity for WebGL at Blue Gravity Studios

In this game I worked writing code on:
• Gameplay systems (movements, enemies, weapons, players, abilities) using Photon Quantum
• UI
