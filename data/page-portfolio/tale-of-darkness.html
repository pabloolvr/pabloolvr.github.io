Tale of Darkness

Tale of Dark is a side scroller, hack & slash roguelite inspired by games like Blasphemous and Dark Souls. The player can traverse dangerous levels filled with various enemies and traps, find new weapons, upgrade skills and unlock new items in order to survive. Developed in Unity for PC at Blue Gravity Studios

In this game I worked writing code on:
  • • Gameplay Combat System using Photon Fusion
    • • Player Stats and Controls mechanics
    • • Weapons
    • • Enemies and Bosses
    • • Spells
    • • Trinkets
    • • Curses
  • • Procedural Level Generation System
  • • Gameplay and Menu UI
Although initially planned as a singleplayer game, there are plans for a coop mode in the future, so the game was written using Photon Fusion architecture.
